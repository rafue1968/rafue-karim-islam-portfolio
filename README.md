
# Rafue Karim Islam's Portfolio

Welcome to my professional portfolio application. This project showcases my work, skills, and experience in full-stack development.

## Overview

A modern, responsive portfolio website built to highlight projects, technical expertise, and professional background.

## Features

- Clean, professional design
- Responsive layout for all devices
- Project showcase with descriptions
- Contact information and links

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js
- React
- CSS/Tailwind CSS (or your preferred styling)
